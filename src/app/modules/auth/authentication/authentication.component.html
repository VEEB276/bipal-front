Banco de información de perfiles académicos y laborales - BIPAL.

Sistema Hoja de Vida para el Talento Local Este sistema tiene como objetivo ampliar las oportunidades laborales para los profesionales de la región mediante la recopilación de información en una base de datos centralizada. Al registrarse en la plataforma y completar el formulario, su perfil quedará disponible para consulta cuando la alcaldía requiera cubrir una vacante en una profesión específica. En ese momento, se revisará la información registrada en la base de datos y se seleccionarán candidatos para participar en un proceso deselección esta iniciativa, buscamos reunir y organizar las ofertas laborales en un único espacio, facilitando la conexión entre talento local y oportunidades de empleo.
